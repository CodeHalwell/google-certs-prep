# Exam Guide – Google Cloud Associate Cloud Engineer

_Last Updated: November 30, 2025_

- **Exam format:** 2 hours, approximately 50–60 multiple‑choice/multiple‑select questions
- **Cost:** Approximately **£99** (about **$125 USD**) – confirm latest pricing
- **Recommended experience:** 6+ months hands‑on with GCP
- **Certification validity:** 3 years

## Exam Domains

- [ ] Set up a cloud solution environment (**17.5%**)
- [ ] Plan and configure a cloud solution (**17.5%**)
- [ ] Deploy and implement a cloud solution (**25%**)
- [ ] Ensure successful operation of a cloud solution (**20%**)
- [ ] Configure access and security (**20%**)

Use `study-schedule.md` for pacing and `core-services/` for core product notes.
