_Last Updated: November 30, 2025_

This track builds a solid foundation in Google Cloud services and operations.

- Role: Cloud engineers, DevOps practitioners, infrastructure specialists

- `networking/` – VPC, load balancing, Cloud DNS
- `storage-databases/` – Cloud Storage, Cloud SQL, Firestore, Bigtable
- `iam-security/` – IAM roles, service accounts, security best practice
- `monitoring-operations/` – Monitoring, Logging, Error Reporting
- `hands-on-labs/` – curated lab list and notes
- `practice-exam.md` – practice questions
